# GitHub User Search App

A modern GitHub User Search application built using **HTML, CSS, and JavaScript** that fetches real-time data from the GitHub API and displays developer profiles in a clean, interactive UI.

This project demonstrates **API consumption, async/await, error handling, loading states, and UI micro-interactions**.

---

## Live Features
- Search GitHub users by username
- Fetch data from the GitHub REST API
